Frequently Asked QNA

Last updated on 29th of April in 2024 (IST).

These are the answers of some frequently asked questions. If you need further help, don't hesitate to join the support server.

  1. How do I invite Deejay?

    Answer: To invite Deejay, ensure you have Manage Server permissions in your server. Then, navigate to the invite link provided here!

  2. Why is the sound laggy and distorted?

    Answer: Laggy and distorted sound is often caused by Discord voice server issues. Try switching your server region to US East for the best connection with Deejay.

  3. Why does the volume change a few seconds late after I adjust it?

    Answer: The delay in volume adjustment is due to the buffer time of the song being played. This delay is intentional and not a glitch or bug.

  4. How can I change Deejay's settings?

    Answer: Explore all available settings commands using the "/help admin" command. Then, use "/command_name" to adjust the desired settings.

  5. Deejay is offline in my server. How can I bring it online?

    Answer: Check our Discord server's announcement channel for any known issues affecting Deejay's availability.

  6. What sources does Deejay support for playback?

    Answer: Deejay can play from over 700 different sites, including YouTube, SoundCloud, Spotify, Vimeo, and more.

  7. How can I view all available commands for Deejay?

    Answer: Simply use the "/help" command in Discord to access the full list of commands.

  8. How do I change the channel for the "start playing" message?

    Answer: Use the "/announce <#new-channel>" command and mention or use it in the desired new channel. Ensure Deejay has necessary permissions in the new channel.

  9. How can I adjust the volume for Deejay?

    Answer: Utilize the "/volume" command to adjust the volume for your current session. Set the default volume using the "/defaultvolume" command.

  10. Is Deejay open source?

    Answer: Yes Deejay's source code is publicly available in this github repository.

  11. Can I block commands in specific channels?

    Answer: While there's no built-in function to block all commands, you can use the blacklist settings to disable music commands in desired channels or adjust server permissions to disable all commands.

  12. What language and libraries does Deejay use?

    Answer: Deejay is written in Node.js (JavaScript) and utilizes multiple libraries.

  13. How does Deejay function?

    Answer: Deejay is a Node.js app built around the discord.js library.

  14. Will I need to manually update Deejay?

    Answer: No, Deejay will be automatically updated.

  15. How is Deejay hosted?

    Answer: Deejay is currently hosted on a VPS located on the east coast of the US.

  16. Can I assist in testing Deejay?

    Answer: Absolutely! You can report bugs and contribute to testing and developing new features on our official Support Server.
